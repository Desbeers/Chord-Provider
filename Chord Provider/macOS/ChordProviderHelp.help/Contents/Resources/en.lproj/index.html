Chord Provider

This is a simple application to view and edit 'chordpro' files for your guitar songs.

I made it because I wanted to have a 'chordpro' viewer/editor for Mac and iPad that is using iCloud for storage. Couldn't find it...

It is released under the GPL3 licence and the project is on GitHub.

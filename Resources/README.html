Chord Provider

A ChordPro viewer, editor and PDF exporter for macOS

What is ChordPro

“A simple text format for the notation of lyrics with chords”.

ChordPro is an open standard text format and its has its own official reference implementation. I contributed a lot to its Open Source code.

While I hope you like Chord Provider, as guitar player or macOS programmer, it is just my hobby project. Go there for the full Chordpro experience.

It is not just drag and drop

Providing applications for the Mac platform is not as easy as it seems. For the users, its mostly just drag and drop. It is not that easy for developers. Apple has a lot of security in place to make sure your drag and drop will be safe for you.

It needs a lot of time, effort and yeh... money for developers to provide such application for the Mac.

Chord Provider is not a commercial product. It is an Open Source application made by me just for fun. You are welcome to download the source, compile it and enjoy!

That is not realistic for must users, especially not on the Mac. I would love to provide you the expected experience but I can’t. I don't have an Apple Developers certificate to provide you this ready to go experience.

The Gatekeeper

Your download contains an almost ready to go Chord Provider; except that the application is only ad-hoc signed by me and not by an approved Apple Developer with a valid certificate. So, the security system on your Mac will give an error when you double-click the application. You have met the Gatekeeper. Depending of the version of macOS you have a different message and have to do different additional handlings to open Chord Provider.

The message will be more or less like this:

Apple could not verify “Chord Provider.app“ is free of malware that may harm your Mac or compromise your privacy.

On macOS versions before 15 (Sequoia) it was common practice to right-click the application and choose Open from the menu. That trick will bypass the gatekeeper and let you open the application.

Unfortunately, that does not work anymore on Sequoia and you have to go trough System Settings:

  1. Try to open the application. A prompt saying “Chord Provider.app“ Not Opened' may appear.
  2. Go to System Settings -> Privacy & Security. Scroll down to find ChordPro, and an option to "Open Anyway".
  3. Choose "Open Anyway".
  4. Authenticate as an administrator.
  5. Chord Provider will now open.

Install with the Terminal

Your download also contains an Install script that can do this for you. Same as with the Chord Provider.app, you can not just double click to run the script. You have to run it manually in the Terminal yourself:

Open the Terminal application in Applications/Utilities and just drag and drop the Install script into its window.

It will do the following:

  • Copy Chord Provider to your applications folder
  • Move Chord Provider out of quarantine so the Gatekeeper will let you start the application with just a double-click.

It will not do anything without your permission but by providing your administration password, you will bypass all Apple’s security measurements.
